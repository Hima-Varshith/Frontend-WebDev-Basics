# Frontend Web Development Basics

This repository contains beginner-level **portfolio-style web projects** developed using **HTML**, **CSS**, and **Bootstrap**.  
Each project demonstrates the core aspects of frontend development - including webpage structure, visual styling, and responsive design using Bootstrap components.

---

## Included Projects

- **Website.html** — Basic portfolio layout built with HTML to structure sections and content.  
- **Website2.html** — Enhanced version using CSS for improved styling and layout control.  
- **Website3.html** — Responsive portfolio webpage built with HTML, CSS, and Bootstrap.

---

### How to View
1. Clone or download the repository.  
2. Open any of the `.html` files in a web browser to preview the designs.
